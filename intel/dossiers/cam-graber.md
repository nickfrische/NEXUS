## TRACKING & VISIBILITY — CURRENT STATE

### Industry Context
- **Smoker Craft Boats** (Sylvan, Smoker Craft, Starcraft) — DOES use RFID. Boats pass through gates at key production stages. Vertical rack storage has gates every few racks, narrows to ~dozen boats per gate.
- **Bennington** — Considered RFID but did NOT implement. Reasons: hardware cost for large gates + ongoing IT maintenance burden

### Alera Today
- No full RFID or GPS tracking system in production
- **Outbound shipping:** Patrick Marine Transport handles all boat transport; they have GPS on loads, but mostly use communication — GPS only when issue/delay arises
- **GPS challenges in-plant:** Metal roofs block GPS (Simrad units confirmed can't get GPS inside); vertical racking causes overlap/confusion

### Alera's Planned System (post-July 2026 move)
- ERP: NetSuite
- Plan: iPad at each production station. Stage complete + QC done → everyone signs off on iPad → stage marked complete → flows into NetSuite
- Sales team (and eventually dealers) can see hull # status: e.g. "Boat 250 is in framing, due off line in a week"
- **Risk identified:** Manually dependent — if an employee doesn't use the iPad, data suffers. Unlike automatic RFID gate scans
